Dalil L'H√¥pital

Pengertian Dalil L'H√¥pital

üìñ Definisi

Dalil L'H√¥pital digunakan untuk menghitung limit yang menghasilkan bentuk tak tentu (indeterminate form) dengan cara membagi turunan pembilang dengan turunan penyebut.

\displaystyle\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}

Berlaku jika limit menghasilkan bentuk \frac{0}{0} atau \frac{\infty}{\infty}

Bentuk-Bentuk Tak Tentu

Bentuk Transformasi
\frac{0}{0} Langsung L'H√¥pital
\frac{\infty}{\infty} Langsung L'H√¥pital
0 \cdot \infty Ubah ke \frac{0}{1/\infty}
\infty - \infty Ubah ke pecahan tunggal
0^0, 1^\infty, \infty^0 Gunakan logaritma natural

Contoh: Bentuk \frac{0}{0}

üìù Soal

Hitung \displaystyle\lim_{x \to 9} \frac{\sqrt{x+7} - 4}{x - 9}

Penyelesaian:

  1. Substitusi langsung: \frac{\sqrt{16} - 4}{0} = \frac{0}{0} ‚úì
  2. Turunan pembilang: f'(x) = \frac{1}{2\sqrt{x+7}}
  3. Turunan penyebut: g'(x) = 1
  4. L'H√¥pital: \lim_{x \to 9} \frac{1}{2\sqrt{x+7}} = \frac{1}{2\sqrt{16}} = \frac{1}{8}
Jawaban: \frac{1}{8}

Garis Singgung

üìñ Definisi

Garis singgung adalah suatu garis yang hanya memiliki satu titik singgung dengan kurva.

m = f'(x_0)

Persamaan:
y - y_0 = f'(x_0)(x - x_0)

Garis Normal

üìñ Definisi

Garis normal adalah garis yang tegak lurus dengan garis singgung di titik singgung.

‚ö° Hubungan Gradien

m_1 \cdot m_2 = -1

Gradien normal: m_2 = -\frac{1}{f'(x_0)}

Contoh: Gradien Garis Singgung

üìù Soal

Tentukan gradien garis singgung y = x^2 - 8x + 12 di titik (1, 5)

Penyelesaian:

  1. Turunan: f'(x) = 2x - 8
  2. Substitusi x = 1:
m = f'(1) = 2(1) - 8 = -6
Gradien = -6

üìê Teorema Kecekungan

üìù Soal

  3. Titik belok: f''(x) = 0 \Rightarrow x = 1

Titik Stasioner

üìñ Definisi

Titik stasioner (titik kritis/titik ekstrim/titik balik) adalah titik pada kurva dengan gradien garis singgung = 0

f'(a) = 0

Uji Titik Stasioner

Uji Turunan Pertama

f': + ‚Üí ‚àíMaks
f': ‚àí ‚Üí +Min
Tidak berubahTitik Belok

Uji Turunan Kedua

Contoh: Titik Stasioner

üìù Soal

Tentukan titik stasioner dari f(x) = 2x^2 - 12x + 20

  1. f'(x) = 4x - 12
  2. f'(x) = 0 \Rightarrow 4x - 12 = 0 \Rightarrow x = 3
  3. f(3) = 2(3)^2 - 12(3) + 20 = 18 - 36 + 20 = 2
  4. Uji: f''(x) = 4 > 0 ‚Üí Minimum
Titik stasioner: (3, 2) ‚Äî Minimum lokal

Nilai Maksimum & Minimum

üìñ Definisi

Misalkan S himpunan tak kosong, dan f mengandung titik c

Nilai Maksimum

f(c) maksimum jika:

f(c) \geq f(x)

untuk semua x \in S

Nilai Minimum

f(c) minimum jika:

f(c) \leq f(x)

untuk semua x \in S

Langkah Mencari Maks/Min

Pada interval tertutup [a, b]

  1. 1 Tentukan turunan f'(x)
  2. 2 Cari titik kritis: f'(x) = 0
  3. 3 Evaluasi f di titik kritis & ujung interval
  4. 4 Bandingkan: terbesar = maks, terkecil = min

Contoh: Nilai Maks & Min

üìù Soal

Tentukan nilai maks & min dari f(x) = x^2 - 6x + 5 pada 0 \leq x \leq 5

  1. f'(x) = 2x - 6 = 0 \Rightarrow x = 3
  2. Evaluasi:
    f(0) = 5
    f(3) = 9 - 18 + 5 = -4
    f(5) = 25 - 30 + 5 = 0
Maks = 5 di x = 0
Min = ‚àí4 di x = 3

üìù Latihan Soal

1. Sketsakan grafik f(x) = \frac{x^2 - 2x + 4}{x - 2}

2. Tentukan nilai limit:

  • a. \displaystyle\lim_{x \to 1} \frac{x^3 - 1}{x^3 - 2x^2 + 2x - 1}
  • b. \displaystyle\lim_{x \to 0} \frac{\ln(\tan x)}{\ln(\tan 2x)}
  • c. \displaystyle\lim_{x \to 0} x \cdot \cot 2x
  • d. \displaystyle\lim_{x \to 0} x^x
