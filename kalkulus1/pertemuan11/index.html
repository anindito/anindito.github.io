üìê Definisi Turunan

\[f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}\]

Definisi ini menjadi dasar untuk menurunkan semua rumus turunan fungsi trigonometri.

‚≠ê Limit Trigonometri Penting

Limit Pertama

\[\lim_{x \to 0} \frac{\sin x}{x} = 1\]

Limit Kedua

\[\lim_{x \to 0} \frac{\cos x - 1}{x} = 0\]

üí° Kedua limit ini sangat penting dalam pembuktian turunan sin x dan cos x!

üî¢ Identitas Trigonometri

IdentitasRumus
\(\sin(A + B)\)\(\sin A \cos B + \cos A \sin B\)
\(\sin(A - B)\)\(\sin A \cos B - \cos A \sin B\)
\(\cos(A + B)\)\(\cos A \cos B - \sin A \sin B\)
\(\cos(A - B)\)\(\cos A \cos B + \sin A \sin B\)

üìò Turunan Fungsi Sinus

Teorema

\[f(x) = \sin x \Rightarrow f'(x) = \cos x\]

Pembuktian Turunan sin x (1/2)

Langkah 1: Definisi turunan
\[f'(x) = \lim_{h \to 0} \frac{\sin(x+h) - \sin x}{h}\]
Langkah 2: Gunakan identitas sin(A+B)
\[= \lim_{h \to 0} \frac{\sin x \cos h + \cos x \sin h - \sin x}{h}\]
Langkah 3: Kelompokkan
\[= \lim_{h \to 0} \frac{\sin x (\cos h - 1) + \cos x \sin h}{h}\]

Pembuktian Turunan sin x (2/2)

Langkah 4: Pisahkan limit
\[= \sin x \lim_{h \to 0} \frac{\cos h - 1}{h} + \cos x \lim_{h \to 0} \frac{\sin h}{h}\]
Langkah 5: Substitusi nilai limit
\[= \sin x \cdot (0) + \cos x \cdot (1)\]
Hasil: \[f'(x) = \cos x \quad \checkmark\]

üìó Turunan Fungsi Cosinus

Teorema

\[f(x) = \cos x \Rightarrow f'(x) = -\sin x\]

‚ö†Ô∏è Perhatikan tanda negatif! Ini adalah kesalahan umum yang sering terjadi.

Pembuktian Turunan cos x

Langkah 1: \[f'(x) = \lim_{h \to 0} \frac{\cos(x+h) - \cos x}{h}\]
Langkah 2: Gunakan identitas cos(A+B)
\[= \lim_{h \to 0} \frac{\cos x \cos h - \sin x \sin h - \cos x}{h}\]
Langkah 3: Substitusi limit
\[= \cos x \cdot (0) - \sin x \cdot (1) = -\sin x\]

üìô Turunan Fungsi Tangen

Teorema

\[f(x) = \tan x \Rightarrow f'(x) = \sec^2 x\]

Karena \(\tan x = \frac{\sin x}{\cos x}\), gunakan aturan hasil bagi.

Pembuktian Turunan tan x

Langkah 1: u = sin x ‚Üí u' = cos x, v = cos x ‚Üí v' = -sin x
Langkah 2: Terapkan aturan hasil bagi
\[\frac{d}{dx}(\tan x) = \frac{\cos x \cdot \cos x - \sin x \cdot (-\sin x)}{\cos^2 x}\]
Langkah 3: Sederhanakan
\[= \frac{\cos^2 x + \sin^2 x}{\cos^2 x} = \frac{1}{\cos^2 x} = \sec^2 x\]

üîó Aturan Rantai

Jika \(u = u(x)\), maka:

FungsiTurunan
\(\sin u\)\(\cos u \cdot \frac{du}{dx}\)
\(\cos u\)\(-\sin u \cdot \frac{du}{dx}\)
\(\tan u\)\(\sec^2 u \cdot \frac{du}{dx}\)
\(\sec u\)\(\sec u \tan u \cdot \frac{du}{dx}\)

‚úèÔ∏è Contoh 1

Tentukan turunan dari: \(y = \sin^4(x^3 + 5)\)

Identifikasi: y = [sin(x¬≥ + 5)]‚Å¥
Aturan rantai:
\(\frac{dy}{dx} = 4[\sin(x^3+5)]^3 \cdot \cos(x^3+5) \cdot 3x^2\)
\(\frac{dy}{dx} = 12x^2 \sin^3(x^3+5) \cos(x^3+5)\)

‚úèÔ∏è Contoh 2

Carilah turunan dari: \(y = \frac{1 - \cos x}{1 + \cos x}\)

Aturan hasil bagi: u = 1 - cos x ‚Üí u' = sin x, v = 1 + cos x ‚Üí v' = -sin x
\(\frac{dy}{dx} = \frac{(\sin x)(1+\cos x) - (1-\cos x)(-\sin x)}{(1+\cos x)^2} = \frac{2\sin x}{(1+\cos x)^2}\)
\(\frac{dy}{dx} = \frac{2\sin x}{(1 + \cos x)^2}\)

‚úèÔ∏è Contoh 3

Carilah turunan dari: \(y = \frac{\sin x \cdot \sec x}{1 + \tan x}\)

Sederhanakan: \(y = \frac{\tan x}{1 + \tan x}\)
Aturan hasil bagi: u = tan x, v = 1 + tan x
\(\frac{dy}{dx} = \frac{\sec^2 x}{(1 + \tan x)^2}\)

‚úèÔ∏è Contoh 4

Carilah turunan dari: \(y = \sqrt{\frac{a\cos^2 x + b\sin^2 x}{\cos^2 x}}\)

Sederhanakan: \(y = \sqrt{a + b\tan^2 x}\)
\(\frac{dy}{dx} = \frac{b\tan x \sec^2 x}{\sqrt{a + b\tan^2 x}}\)

‚úèÔ∏è Contoh 5

Carilah turunan dari: \(y = \frac{2\sin^3 2x}{\cos^2 2x}\)

Gunakan aturan hasil bagi dengan u = 2sin¬≥(2x), v = cos¬≤(2x)
\(\frac{dy}{dx} = \frac{4\sin^2(2x)(2 + \cos^2(2x))}{\cos^3(2x)}\)
