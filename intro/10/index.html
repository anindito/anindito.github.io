Apa itu Botnet?

Botnet = Jaringan komputer "zombie" üßü

  1. Penyerang menyebarkan malware ke banyak komputer
  2. Komputer terinfeksi menjadi "bot" atau "zombie"

  4. Penyerang mengendalikan semua bot dari jarak jauh
  5. Semua bot menyerang target secara bersamaan

Apa itu "Session"?

Seperti tiket masuk perpustakaan. Anda tunjukkan kartu anggota saat masuk, dapat tiket. Setelah itu, cukup tunjukkan tiket untuk akses semua area. Tidak perlu tunjukkan kartu lagi.

Session = "Ingatan" website tentang siapa Anda

  • Session ID disimpan di browser (cookie)
  • Setiap request otomatis kirim Session ID

Cara Melindungi Diri

Defense in Depth

Melindungi dari DoS/DDoS

Gunakan strategi berlapis! (Defense in Depth)
  • Rate Limiting: Batasi jumlah request per IP
  • Traffic Filtering: Blokir traffic mencurigakan
  • Load Balancing: Distribusi beban ke banyak server
  • CDN: Content Delivery Network untuk caching
  • DDoS Protection Service: Cloudflare, Akamai, AWS Shield
